Aide - EasyMap

Comment utiliser EasyMap

Pour générer une carte mentale, suivez ces étapes :

  1. Cliquez sur le bouton "Générer une carte" dans la page d'acceuil.
  2. Sélectionner le cours que vous souhaiter transformer en ume carte mentale.
  3. Choiser le modèle qui vous convient.
  4. La carte mentale générée automatiquement s'affiche.
  5. Modifier votre carte en utilisant les outils qui se trouvent dans la section Modifier.
  6. Ajouter une image ou/et une note en utilisant les outils qui se trouvent dans le haut de la page.
  7. Sauvegarder la carte mentale dans votre machine cliquant sur Sauvegarder puis sélectionnant le format PNG ou PDF.
  8. Partager la carte mentale sur gmail ou/et google drive en cliquant sur Partager.

Pour créer une carte mentale, suivez ces étapes :

  1. Cliquez sur le bouton "Créer une carte" dans la page d'acceuil.
  2. Ouvrir le cours pour lequel vous souhaitez créer une carte mentale.
  3. Modifier votre carte en utilisant les outils qui se trouvent dans la section Modifier.
  4. Ajouter une image ou/et une note en utilisant les outils qui se trouvent dans le haut de la page.
  5. Sauvegarder la carte mentale dans votre machine cliquant sur Sauvegarder puis sélectionnant le format PNG ou PDF.
  6. Partager la carte mentale sur gmail ou/et google drive en cliquant sur Partager.
