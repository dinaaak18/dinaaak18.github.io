Comment utiliser EasyMap

Pour créer une carte mentale, suivez ces étapes :

  1. Cliquez sur le bouton "Nouvelle carte" dans la barre de navigation.
  2. Utilisez les outils "Ajouter un nœud" et "Ajouter une relation" pour ajouter des éléments à votre carte.
  3. Cliquez sur le bouton "Enregistrer la carte" pour enregistrer votre travail.

Raccourcis clavier

EasyMap propose les raccourcis clavier suivants :

  • Ctrl + N : créer une nouvelle carte
  • Ctrl + S : enregistrer la carte actuelle
  • Ctrl + Z : annuler la dernière action
  • Ctrl + Y : rétablir la dernière action annulée
